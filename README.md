This project focuses on analyzing Blinkit retail sales data to uncover insights related to product performance, outlet efficiency, customer preferences, and sales trends.

The visualization was built to replicate a real-world retail analytics use case, commonly encountered in Business Analyst, Data Analyst, and Consulting roles.
